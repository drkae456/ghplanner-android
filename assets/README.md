# Assets

Place production-ready assets before shipping builds:

- `icon.png` – 1024x1024 PNG (Google Play icon source).
- `adaptive-icon.png` – Foreground layers (1024x1024) with transparent background.
- `splash.png` – 1242x2436 portrait splash (safe area 1125x2436).
- `favicon.png` – 512x512 for web preview.

These placeholders are ignored by the build system until real files are added.
